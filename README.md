# Remote-pair-programming
It is an online platform to write and share code in real time. Some of the key functionalities are
- run C, C++, Python programs 
- login with github or gmail id
- pair program with your friend and can have chat/video-chat at the same time.

# Run the application
To run the application, you will need latest node.js and mongoDB installed in your system. Then do the following steps:
- run ```npm i``` in cmd in the project root directoy
- run ```npm run dev``` you will see server running at port 4001
For Database, you have to create a mongoDB database named "remotePP" and two collections (take reference from models/<>.js).
